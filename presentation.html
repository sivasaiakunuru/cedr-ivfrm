🔒 Security Architecture

🛡️ Cryptographic Protections

  • AES-256-GCM: Hardware-accelerated encryption for data at rest
  • SHA-256 Hash Chain: Blockchain-style tamper-evident logging
  • TLS 1.3: Latest transport security for data in transit
  • NXP A71CH HSM: FIPS 140-2 Level 2 secure key storage
  • Certificate Pinning: Prevents MITM attacks on vehicle-cloud link

⚠️ Physical Security

  • Enclosure Breach: Immediate detection of unauthorized access
  • Temperature Monitoring: Detects environmental tampering
  • Anti-Tamper Mesh: Circuit triggers zeroization on breach
  • Secure Boot: Verified boot chain with signed firmware
  • Zero-Trust: Continuous verification architecture
CEDR provides court-admissible forensic evidence with cryptographic proof of integrity—essential for insurance claims and regulatory compliance.
